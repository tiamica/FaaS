# Fourier Analytics AI Search (FaaS)

The most intelligent search engine for Africa, focusing on positive business insights and high-value opportunities across the continent.

## Features

- **AI-Powered Search**: Intelligent responses about African business and opportunities
- **Country-Specific Insights**: Detailed information about African nations
- **Positive Affirmations**: Focus on growth, innovation, and success stories
- **Clean Interface**: Modern, responsive design with excellent UX
- **Real-time Suggestions**: Quick access to popular search topics

## Technology Stack

- **Frontend**: Vanilla JavaScript with ES6+ modules
- **Styling**: Modern CSS with CSS Grid and Flexbox
- **AI Integration**: Configurable for free LLM APIs (Hugging Face, Together AI, etc.)
- **Build Tool**: Vite for fast development and building
